# An open source OptiX 7 based SAR simulator

Uses ray tracing for SAR simulation. Runtime is reduced by utilizing the NVIDIA OptiX raytracing engine.

# Requirements

* x64 build system
* linux (windows not tested)
* conan
* git
* cmake version >= 3.18
* C++20 compliant compiler
